# Image Guide

The 20 product images in `assets/images/products/` are **AI-generated studio product photos**
(created with the Bria API — royalty-free), giving a clean, consistent catalogue out of the box.
They are PNG files named after each product, e.g. `football-ball.png`, `running-shoes.png`.

## Replacing a product image (optional)
The easiest way is **Admin → Products → Edit → Product Image** — upload any JPG/PNG and the
system stores it and points the product at it automatically. You can also drop a file directly
into `assets/images/products/` and set the product's `image` field to that filename.

The home-page hero uses a clean dark banner (no image file required).
